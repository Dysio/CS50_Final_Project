Home Page

Welcome to DYSIO PDF EDITOR where you can edit your pdf files with some simple functions.

Merge

With merge you can join multiple pdf files. They are joined in alphabeticall way so remember to named the files first and then upload them.

Split

With split you can split the whole document just by uploading the file and passing numbers of pages where you want to split the file.

Split By Page Size

With split by page size you can split the whole document by page size. The document will be splitted into files that consist one height of page: A4, A3, 297, 420, 610 and bigger.

Rotate

With rotate you can rotate each size of document just by uploading the file and passing numbers of pages where you want to rotate the page.
